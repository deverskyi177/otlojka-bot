# Sialens — Telegram-бот модерации, аналитики и развлечений

Бот на **aiogram 3**, SQLite (создаётся автоматически), интеграция с **Gemini API** и оплата подписки через **Telegram Stars**.

## Функционал

- **Модерация**: `.mute 1ч|6ч|12ч|24ч|навсегда`, `.warn`, `.ban`, `.unban`, `.kick`, `.clean [N]`, `.wbl` (фильтр мата), `.wsag` (фильтр стикеров/гифок)
- **Фильтры**: капс (>70%), ссылки, флуд (3+ сообщения за 5 сек → автомут)
- **Авто-модерация**: 3 варна = бан на 24ч
- **ИИ**: `.ai <текст>` с контекстом последних 15 сообщений чата (Gemini)
- **Игры**: `.revo`, `.anim 1-5`, `.guess`, `.rps`, `.games`
- **Аналитика**: `.stats`, `.top`, `.nightowl`, `.peak`
- **Триггеры**: `.addtrigger "ключ" "ответ"`, `.removetrigger "ключ"`, `.trigger list`
- **Опросы**: `.poll "вопрос" "вар1" "вар2" ...`
- **Напоминания**: `.remind "текст" 10м` (с/м/ч/д)
- **Раскладка**: `.sw` — переключение раскладки ru↔en
- Авто-приветствие и прощание с новыми/ушедшими участниками
- **Роли**: новичок (0–50 XP) → проверенный (50+ XP) → VIP (подписка) → админ
- **Подписка**: триал 3 дня, далее 50 ⭐/мес (Telegram Stars). Платно: ИИ, аналитика, триггеры
- **Админ-панель**: `/admin` — инлайн-меню (фильтры, статистика, подписка, триггеры)
- **Команды супер-админа бота**: `/givepremium`, `/removepremium`, `/addadmin`, `/removeadmin`, `/banuser`, `/unbanuser`, `/setprice`, `/settrial`, `/resetwarns`, `/resetstats`

## Установка локально

1. Установите Python 3.11+.
2. Склонируйте/скопируйте файлы проекта в отдельную папку.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Скопируйте `.env.example` в `.env` и заполните переменные:
   ```bash
   cp .env.example .env
   ```
   - `BOT_TOKEN` — получите у [@BotFather](https://t.me/BotFather)
   - `GEMINI_API_KEY` — получите на [Google AI Studio](https://aistudio.google.com/app/apikey)
   - `ADMIN_IDS` — ваш Telegram id (узнать у [@userinfobot](https://t.me/userinfobot)), через запятую, если несколько
5. Запустите бота:
   ```bash
   python bot.py
   ```
   При первом запуске автоматически создадутся папки `data/`, `logs/` и файл базы `data/sialens.db`.

## Настройка бота в BotFather

1. Отключите Privacy Mode (`/setprivacy` → `Disable`), чтобы бот видел все сообщения в группе, а не только команды.
2. Включите инлайн-режим не требуется.
3. Добавьте бота в чат и выдайте ему права администратора (обязательно: удаление сообщений, бан/мут участников, закрепление — по желанию).

## Деплой на Railway

1. Создайте новый проект на [railway.app](https://railway.app) → **Deploy from GitHub repo**, указав репозиторий с этими файлами (либо загрузите файлы через Railway CLI).
2. В настройках проекта откройте вкладку **Variables** и добавьте:
   - `BOT_TOKEN`
   - `GEMINI_API_KEY`
   - `ADMIN_IDS`
3. Railway автоматически определит `requirements.txt` и `Procfile` и запустит процесс `worker: python bot.py`.
   - Если Railway предложит создать веб-сервис — выберите тип **Worker** (не нужен открытый HTTP-порт, бот работает через long polling).
4. База данных SQLite хранится в файловой системе контейнера (`data/sialens.db`). На бесплатном плане Railway диск эфемерный при пересборке — для постоянного хранения подключите **Railway Volume** и примонтируйте его в `/app/data`.
5. После деплоя проверьте логи (`Deployments → View Logs`) — там же дублируется всё, что пишется в `logs/bot.log`.

## Примечания по оплате (Telegram Stars)

- Оплата подписки реализована через нативный Telegram Stars (`currency="XTR"`), отдельный платёжный провайдер не требуется.
- Цена и длительность триала настраиваются командами `/setprice` и `/settrial` (только супер-админы бота из `ADMIN_IDS`/`/addadmin`).

## Роли и права

| Роль | Условие |
|---|---|
| Новичок | 0–49 XP |
| Проверенный | 50+ XP (1 XP = 1 сообщение) |
| VIP | активная подписка (триал или Stars) |
| Админ | указан в `ADMIN_IDS` или добавлен через `/addadmin` |

Команды `.mute`, `.warn`, `.ban`, `.unban`, `.kick`, `.clean`, `.wbl`, `.wsag` доступны администраторам **чата** (Telegram-права admin/creator) или админам бота.
Команды `.ai`, `.stats`, `.top`, `.nightowl`, `.peak`, `.addtrigger`, `.removetrigger` требуют активной подписки VIP (либо статуса админа бота).
