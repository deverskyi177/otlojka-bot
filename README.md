# Sialens — учебный Telegram-бот на aiogram 3

Этот репозиторий содержит учебный бот Sialens — модератор, помощник, с простыми ИИ-функциями и играми. Проект настроен для деплоя на Railway.

Внимание: это учебный пример. Некоторые функции (платежи, Gemini) требуют реальных ключей и настройки.

## Файлы
- `bot.py` — основной файл с кодом бота (весь функционал).
- `requirements.txt` — зависимости.
- `Procfile` — команда запуска для Railway.
- `.env.example` — пример переменных окружения.
- `README.md` — инструкция.

## Переменные окружения
Создайте `.env` на основе `.env.example` или задайте переменные в Railway:
- `BOT_TOKEN` — токен Telegram-бота.
- `GEMINI_API_KEY` — ключ Google Gemini (опционально, для команды `.ai`).
- `ADMIN_IDS` — список админов через запятую (например `12345678,87654321`).

## Запуск локально
1. Клонируйте репозиторий.
2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
3. Создайте `.env` и заполните переменные.
4. Запустите:
   ```
   python bot.py
   ```

При первом запуске создаётся база данных `sialens.db` и папка `logs/` с `bot.log`.

## Деплой на Railway
1. Создайте проект на Railway и подключите репозиторий.
2. В настройках проекта добавьте переменные окружения (`BOT_TOKEN`, `GEMINI_API_KEY`, `ADMIN_IDS`).
3. Перейдите в раздел Deploy и выберите `Deploy`.
4. Railway автоматически установит зависимости и запустит `Procfile` (`worker: python bot.py`).

## Основной функционал
- Модерация: `.mute`, `.warn`, `.ban`, `.unban`, `.kick`, `.clean`.
- Фильтры: капс (>70%), ссылки, стикеры/гифки, мат — все базовые фильтры.
- Авто-модерация: 3 варна → бан 24ч.
- ИИ: `.ai <текст>` — использует Gemini API (включите `GEMINI_API_KEY`).
- Игры: `.revo`, `.guess`, `.rps`.
- Триггеры: `.addtrigger "ключ" "ответ"`, `.removetrigger "ключ"`, `.trigger list`.
- Опросы: `.poll "вопрос" "вариант1" "вариант2" ...`
- Напоминания: `.remind "текст" 10м`
- Перевод раскладки: `.sw`
- Авто-приветствие и прощание при входе/выходе.
- Ролевые и премиум функции (упрощённо) и обработка успешных платежей.
- Админ-панель через инлайн-кнопки (`/start` -> "Админ-панель").
- SQLite база (`sialens.db`) создаётся автоматически.
- Логи в `logs/bot.log`.

## Примечания и что можно улучшить
- Платежи: для реальных платежей необходимо настроить провайдера и обработку invoice.
- AI: убедитесь, что используемая версия `google-generativeai` соответствует API Google и у вас есть права.
- Улучшить проверку упоминаний/парсинг команд, обработку исключений и права доступа.
- Добавить тесты, миграции БД и более устойчивое управление задачами (например, APScheduler).

Если нужно, могу:
- Разбить код на модули;
- Добавить более точную обработку ролей и подписок;
- Реализовать полноценную панель управления в веб-интерфейсе;
- Помочь настроить оплату через Telegram (Stars/Payments) под вашу страну.
